# SOOI
Scenario-based Optimization of Outlying Interchanges (SOOI) - is a method of analysis and further problem solving for interchanges located at the connections of the main city highways with peripheral urban development

Developed as the result of Master's degree final qualifying work which is 'Development of a methodology for optimizing transport interchanges in peripheral urban development'
2026, ITMO University, Institute of Design and Urban Studies

This repository is a set of SUMO configs that simulate different optimization scenarios for two interchanges located in the east part of Saint-Petersburg, Russia
